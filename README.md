# dotfiles

## Usage

```bash
$ git clone git@github.com:takeyarise/dotfiles.git
$ cd dotfiles
```

**to install**
```bash
$ cd dotfiles
$ stow -v <dir>
$ stow -v vim
```

**to remove**
```bash
$ cd dotfiles
$ stow -vD <dir>
$ stow -vD vim
```
